下载python

  • https://www.python.org/downloads/windows/

  • 选择windows版本

  • 下载64位安装包

  • 安装

  • 找到自己安装python地址的根目录，复制地址

  • 配置环境变量，查看path里面有没有该地址，如果没有就将复制的地址粘贴进去点击确定

  • 输入python验证是否安装成功

安装youtube-dl视频下载器

  • pip install youtube-dl                     #安装
  • pip install --upgrade youtube-dl                    #升级

下载ffmpeg

  • https://ffmpeg.zeranoe.com/builds/
  • 下载之后解压，打开到bin目录，复制地址
  • 配置环境变量，将复制的地址粘贴进去
  • 打开cmd，输入ffmpeg -version 查看是否配置成功

开始下载视频

  • youtube-dl --list-formats 想要下载的网址                    #查看该网址视频所有格式
  • youtube-dl -f 315+140 想要下载的网址                    #下载该规格 315=视频分辨率，140=音频，选择自己需要的视频格式+音频格式
  • ffmpeg -i 已下载的视频文件 -vcodec copy -an 命名.mp4
  • ffmpeg -i 命名.mp4 -i 已下载的音频文件 新命名.mp4
    #下载之后是视频和音频分开的文件，使用命令合并即可
  • ffmpeg -i C:\Users\lz\Desktop\aa.webm -vcodec h264 C:\Users\lz\Desktop\aa.mp4                    #H264视频格式转换
  • ffmpeg -i 已下载的视频文件 -vcodec h264 转换后的视频文件地址.mp4
    #可以直接把视频文件拖入命令行里面，避免输入错误

更新视频下载器

  • pip install --upgrade youtube-dl
